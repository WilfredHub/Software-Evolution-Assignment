A simple Task Manager application developed for the Software Evolution course assignment.
The system allows users to:
- Add tasks
- View tasks
- Search tasks
- Mark tasks as completed 

## Installation

1. Download project files.
2. Open in your preferred IDE.
3. Run the application.

## Usage

- Create tasks.
- View existing tasks.
- Search for tasks.
- Update task status.
